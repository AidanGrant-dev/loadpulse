OAuth Integration Guide

Each platform uses OAuth 2.0. Clicking "Connect" will open the provider's authorization page in a new tab. After granting access, paste the authorization_code returned in the URL back here.

StatSports: Register at developer.statsports.com → create an App → use Client ID + Client Secret.
Strava: Create an app at strava.com/settings/api → Authorization URL: https://www.strava.com/oauth/authorize
Whoop: Apply for API access at developer.whoop.com → Authorization URL: https://api.prod.whoop.com/oauth/oauth2/auth

In a production deployment, token exchange should happen server-side to protect your client_secret. This app stores access tokens locally in your browser.
